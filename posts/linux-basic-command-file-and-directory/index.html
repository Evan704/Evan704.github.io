[Linux基础命令] 文件与目录管理

July 1, 2025

导航#

  • pwd: 输出所在的目录路径。
  • ls: 列出当前目录下的文件和文件夹。（详细内容）
  • cd: 切换目录。
    • cd /var/log: 切换到/var/log目录（绝对路径）。
    • cd Documents: 切换到当前目录下的Documents子目录（相对路径）。
    • cd ..: 切换到上一级目录。
    • cd ~: 切换到当前用户的主目录。
    • cd -: 切换到上一次所在的目录。

文件与目录的创建与删除#

  • touch: 创建文件。
    • 例：touch mynote.txt
  • mkdir: 创建目录。
    • 例：mkdir myfiles
  • cp: 复制文件或目录。
    • 例：
      • cp source.txt destination.txt (复制文件)
      • cp -r sourcedir destdir (复制目录，-r代表递归)
  • mv: 移动或重命名文件或目录。
    • 例：
      • mv oldname.txt newname.txt (重命名)
      • mv file.txt ../ (移动到上级目录)
  • rm: 删除文件或目录。
    • 例：
      • rm file.txt: 删除文件。
      • rm -r directory: 删除目录及其下的所有内容。
      • rm -f file.txt: 强制删除，不提示。
      • rm -rf directory: 强制递归删除目录。

查看文件内容#

  • cat: 显示文件内容。
  • less: 分页显示文件内容。
    • 操作：按q退出，/后跟关键词进行检索。
  • head/tail: 查看文件的前几行或后几行。
    • 例：
      • head -n 20 file.txt: 查看文件头20行。
      • tail -n 50 file.txt: 查看文件末尾50行。
      • tail -f /var/log/auth.log: 实时动态显示文件新增的内容。

权限管理#

  • chmod: 修改文件或目录的权限。
    • 例：
      • chmod +x script.sh: 为所有用户添加执行权限。
      • chmod 755 script.sh: (数字模式) 设置权限为 rwxr-xr-x。
  • chown: 修改文件或目录的所有者和/或所属组。
    • 例：
      • chown user:group file.txt

搜索与过滤#

  • find: 在指定目录中搜素文件。
    • 例：
      • find / -name "*.conf": 从根目录开始，查找所有以.conf结尾的文件。
      • find /tmp -user root: 在/tmp目录中查找所有者是root的文件。
  • grep (Global Regular Expression Print): 从文件或输入中筛选包含特定模式的行。
    • 例：
      • grep "Failed password" /var/log/auth.log
  • 管道 | 与 重定向 > >>:
    • 管道 |: 将前一个命令的输出，作为后一个命令的输入。
    • 重定向 >: 将命令的输出覆盖写入到一个文件中。
    • 重定向 >>: 将命令的输出追加到一个文件的末尾。
    • 例：
      • history | grep "ssh" (在历史命令中，筛选出所有包含"ssh"的命令)
