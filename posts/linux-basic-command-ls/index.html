ls（List）命令的作用是列出当前目录下的文件和文件夹。

常用参数#

  • ls -l: 显示详细信息（权限、所有者、大小、修改时间）。
  • ls -a: 显示所有文件，包括以.开头的隐藏文件。
  • ls -al: 二者结合。

文件类型与权限硬链接数所有者所属组大小（bytes）最后修改时间文件名
drwxr-xr-x2kalikali4096JunDesktop

后五列比较好理解，下面来解释一下前两列。

第一列：文件类型与权限#

  • 第一个字符：文件类型
    • -: 普通文件。如文本文件、二进制程序、图片等。
    • d: 目录 (Directory)。
    • l: 符号链接 (Symbolic Link)。类似于Windows的快捷方式。
    • c: 字符设备文件 (Character Device)。如终端 (/dev/tty)。
    • b: 块设备文件 (Block Device)。如硬盘 (/dev/sda)。
  • 后九个字符：权限 这9个字符被分为三组，每组3个字符，分别代表：
    1. 所有者的权限
    2. 所属组的权限
    3. 其他用户的权限 在每一组中，三个字符r,w,x分别代表：
    • r: 读权限。
    • w: 写权限。
    • x: 执行权限。
    • -: 没有相应权限。 对于目录，它们的含义有点不同：
    • r: 可以 ls 这个目录，查看目录下的文件名列表。
    • w: 可以在该目录下创建、删除、重命名文件。
    • x: 可以 cd 进入该目录。
    • -: 没有相应权限。

第二列：硬链接数#

  • 对于文件：表示有多少个文件名指向这个文件。通常是1。如果创建硬链接，这个数字会增加。
  • 对于目录：这个数字表示该目录下包含多少个子目录。起始值是2，代表.（当前目录）和..（上级目录）。每在其中创建一个新目录，该数字就加1。
